# 投资系统

## 系统使命

围绕现金流、风险承受能力，以及“国内可投资资产 + 国际技术前沿锚点”的双层框架，建立可解释、可复盘的投资观察与决策支持系统。

## 当前主线

- 算力
- 电力 / 能源
- 数据中心基础设施
- AI 应用与具身智能

## 运行流程

```text
观察标的 -> 读业务与数据 -> 理解产业链位置 -> 风险估计 -> 决策支持 -> 事后复盘
```

## 新手优先级

先学三件事：

1. 看懂公司靠什么赚钱
2. 看懂增长来自哪里
3. 看懂估值高低在定价什么预期

## 为什么要这样分

因为**国际最前沿很多并不在二级市场直接可买**，尤其具身智能里最强的几家公司大多还是私有公司。对投资系统来说，更合理的做法是：

$$
\text{投资观察} = \text{国内可交易映射资产} + \text{国际技术前沿参照系}
$$

也就是说，先在中国市场里找真正能买、能跟踪、能看财报的公司，再用国际前沿判断它们是“真接近前沿”，还是“只蹭概念”。

## 具身智能要看哪几层

1. 整机平台：优必选、越疆
2. 灵巧手 / 末端执行器：兆威机电
3. 关节 / 减速器 / 丝杠：绿的谐波等
4. 传感 / 力控 / 触觉：他山科技、柯力传感等
5. 大脑 / VLA / 数据引擎：更多依赖国际锚点比较
