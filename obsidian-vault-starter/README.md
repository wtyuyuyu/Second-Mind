# Second Mind Obsidian Starter

这是一个给 Obsidian 使用的最小四系统起步骨架。

目标不是一次把系统做满，而是先让下面这条链路跑起来：

```text
输入 -> 判断 -> 页面沉淀 -> 行动 -> 复盘
```

## 目录

- `00-Dashboard/`：首页与总览
- `10-Research/`：研究系统
- `20-AI/`：AI 系统
- `30-Life/`：生活系统
- `40-Invest/`：投资系统
- `90-Review/`：复盘页面
- `99-Templates/`：模板页

## 用法

1. 在 Obsidian 新建一个 vault
2. 把本目录内容复制进 vault 根目录
3. 安装最小插件集：
   - `Zotero Integration`
   - `Dataview`
   - `Tasks`
   - `Templater`
4. 优先从 `00-Dashboard/Second Mind 首页.md` 开始

## 结构原则

- 每个系统先只有一个总控页
- 高价值输入先变成结构化页面
- 行动项统一写成任务
- 复盘问题固定，不跟着当天情绪漂移
