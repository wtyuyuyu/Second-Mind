        Zotero 与 Obsidian 的最小可运行配置

        Zotero

        负责文献、批注、高亮、引用键与 PDF 注释，是研究系统最上游的高价值输入层。

        • Better BibTeX：生成并维护 citation key，给 Obsidian 引文导入提供稳定锚点。

        Obsidian

        负责本地 Markdown 记忆层、结构化查询、任务聚合与模板化落库。

        • Zotero Integration：导入 citations、bibliography、notes 与 PDF annotations。
        • Dataview：把四系统页面变成可查询的表格、列表与聚合视图。
        • Tasks：统一管理行动项与复盘项。
        • Templater：把研究页、投资页、复盘页做成可重复模板。
        • Local REST API（可选）：后续让 Codex 通过本地 API 安全读写 vault。
